# Sorry Website (GitHub-Ready)

A fully runnable apology webpage made with plain HTML, CSS, and JavaScript.

## Quick Start

1. Download this folder or clone your repo.
2. Open `index.html` directly in your browser.

No build tools or dependencies are required.

## Upload To GitHub

1. Create a new repository on GitHub.
2. Upload all files from this folder to the repository root:
   - `index.html`
   - `style.css`
   - `script.js`
3. Commit and push.

## Publish With GitHub Pages

1. Go to your repo `Settings` -> `Pages`.
2. Under `Build and deployment`:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` (or your default branch), folder `/ (root)`
3. Save.
4. Wait about 1-2 minutes, then open your GitHub Pages URL.

## Customize Text

Edit text inside:
- `index.html` (headings/messages/buttons)

## Notes

- This version is static and GitHub Pages friendly.
- It reproduces the same multi-step interaction flow in a dependency-free way.
